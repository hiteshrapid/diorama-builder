description: Set up a 3D Diorama office for your OpenClaw agents — design the space, wire up events, watch them live
---

Run the `diorama-setup` skill to set up your 3D office.

The skill will:
1. Launch the Diorama builder in your browser so you can design the office (rooms, furniture, themes)
2. Scan your `~/.openclaw/` workspace to understand your agents
3. Interview you about your agent flow and propose event emissions
4. Write the integration code directly into your agent files (with a diff preview)
5. Register the Diorama MCP inside your OpenClaw config
6. Open the live view so you can run your agents and watch them animate in 3D

Invoke the `diorama-setup` skill now.
